_mm_cvtsi32_ss#

Tech:

SSE_ALL

Category:

Convert

Header:

xmmintrin.h

Searchable:

SSE_ALL-Convert-XMM

Register:

XMM 128 bit

Return Type:

__m128

Param Types:

__m128 a, int b

Param ETypes:

FP32 a, SI32 b

__m128 _mm_cvtsi32_ss(__m128 a, int b);

Intel Description

Convert the signed 32-bit integer “b” to a single-precision (32-bit) floating-point element, store the result in the lower element of “dst”, and copy the upper 3 packed elements from “a” to the upper elements of “dst”.

Intel Implementation Psudeo-Code

dst[31:0] := Convert_Int32_To_FP32(b[31:0])
dst[127:32] := a[127:32]
